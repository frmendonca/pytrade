# Pytrade

Pytrade is a portfolio management, simulator and investment strategy helper.

**Stack:** Python, pandas, Jupyter notebooks.

---

## Repository Layout

```
pytrade/
  data/               # External datasets to pad realtime extraction for longer datasets
  pytade/
    analysis/         # Analysis notebooks
    data_models/      # Main classes defining the large blocks for simulation, strategies and portfolio definition
    integrations/     # Client integrations for data extraction
    simulation/       # Main simulator tools
    utils/            # Util methods used throughout the repository
    tests/            # Test methods for functionalities in the repositgitory
```
